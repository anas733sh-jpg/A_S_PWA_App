تجهيز التطبيق الاحترافي — تعليمات التشغيل
1) فك الضغط وادخل المجلد
2) ثبت الاعتمادات: npm install
   (إذا أردت ضغط الصور: npm install sharp --save)
3) شغّل السيرفر: npm start
4) افتح المتصفح على http://localhost:3000
